# TechmartAPI

Для работы API, требуется запустить отладку в VS и открыть любой интересущий html файл.

API работет с базой данной и все новые заказы появляются в БД.

Система рекомендаций рекомендует товары на основе уже добавленных товаров в корзине: для каждого товара ищутся похожие и покупаемые вместе, они ранжируются по рейтингу, и выдаются лучшие; если корзина пуста, показываются популярные товары.

Для каждого авторизированного пользователя система рекомендаций составляет индивидуальные рекомендации товаров, на основе раннее покупаемых товаров

Чтобы зайти в админ панель, необходимо пройти авторизация: логин - admin1 пароль - adminpass.

Администратор имеет возможность добавлять и удалять записи в БД.
